## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/jonathanbitton/Wavelets-WAI/python.git
   cd python
2. install dependencies:
   ```bash
   pip install -r numpy, pandas, scipy, matplotlib, pyfftw
